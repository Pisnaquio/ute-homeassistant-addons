# UTE Home Assistant Add-ons

Repositorio de instalación del add-on estable **UTE** para Home Assistant.

## Instalación

1. En Home Assistant, abrí **Settings > Apps**.
2. Agregá `https://github.com/Pisnaquio/ute-homeassistant-addons` como repositorio personalizado.
3. Instalá **UTE** y configurá tu usuario o número de cuenta de UTE y contraseña.
4. Iniciá el add-on y abrí la UI desde la barra lateral.

## Privacidad

El paquete no incluye credenciales, cuentas, datos de consumo ni datasets de ejemplo del autor. Los datos de cada usuario quedan en el almacenamiento persistente de su propia instancia Home Assistant.
