# UTE Dev

Canal de pruebas del app local de Home Assistant para hostear la UI completa de
UTE dentro de Home Assistant mediante Ingress.

Incluye:

- Dashboard web embebido en la sidebar como `UTE Dev`
- Descarga y refresco de datos desde UTE usando Node + Playwright
- Persistencia separada en `/data/ute-dev`
- Configuración de credenciales desde las opciones del app
